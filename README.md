# Ejercicios de Procesamiento en Streaming

## Procedimiento

### Clonar repositorio en local

- Desde Visual Studio Code, clonar el repositorio desde Clone Git Repository
- Introducir la URL del repositorio de GitHub https://github.com/universidadcomillas/procesamiento-streaming-2026.git

### Envío de ejercicios resueltos

Se habilitará una sección en el moodle para que podáis subir vuestros ejercicios resueltos.  

### Update de ejercicios nuevos

Cuando así lo indique el profesor, ejecuta el comando siguiente para descargar nuevas prácticas:

```bash
git pull --no-edit origin master
```
